### Анализ ошибочных значений давления в КИС
На добывающих скважинах оснащенных ДД провели сравнение значений давления из КИС, которые заносил оператор ДНГ, и значения давлений с ДД, которые передаются на верхний уровень в АСУ. По каждой скважине был сделан срез за октябрь 2025 года. В КИС выбиралась скважина, оснащенная ДД и наличем зданных на верхнем уровне. Далее в срезе по каждой скважине выбирали 4 дня с равной периодичностью в 7 дней и дневной обход, такой выбор и периодичность обусловлена 7 дневным вахтовым методом работы для максимального охвата различных операторов  и светлое время, то есть для снижения человеческого фактора. При выборе значения давления в КИС также регистрировалось время, в которе оператор ДНГ фиксировал значение
